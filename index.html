Images, src, href and the importance of anchor <a> elements

The image element <img> is a form of void element, so no closing tag is needed. Using <img>, you can attach pictures right to the website. The source attribute (ex: img src, source src, href src, audio src, video src, etc) links to a specific file, webpage, or media source. You can tell what type of <src> will be used based on the element name that preceeds it.

Anchor elements <a> allow you to link to another webpage. href means hypertext reference, and this attribute guides to a specific hyperlink location. You can also include text between the opening and closing tags to write your own text, which will give the user an idea of where the text will lead to. Additionally, more written text can be written before and after the anchor element!

HTML Entities - How the heck do you type < and > in HTML?

When coding in html, < and > will not show up on a webpage unless HTML entities are used. This is a named character reference. In fact, if you look at the html file of this document, you will see many elements and notes that are not visible on the webpage. Named references start with an ampersand sign and end with a semicolon. By using a named character reference, the HTML reader will not confuse this with an actual HTML element.

Additionally, you can use < or < which are examples of using the decimal numeric and hexadecimal numeric references, respectivly.

What's the difference: id vs. class?

id

The HTML id attribute is used to specify a unique id for an HTML element. You cannot have more than one element with the same id in an HTML document.The id attribute is used to point to a specific style declaration in a style sheet. It is also used by JavaScript to access and manipulate the element with the specific id. The id attribute CANNOT use spaces, unlike the class attribute.

audio and video

Audio and video elements work similarly and are fairly simple to code. To add an audio player, use the controls attribute. To loop music, use the loop attribute. To start music muted, use the muted attribute (this on/off nature of muted state is a good exanple of a boolean attribute).

Depending on browsers, some media players won't be supported, so it's a good idea to add a few sourse elements for browsers to pick from in case the first or second optiob fails. Code this as:

<source src="audio.ogg" type="audio/ogg"

<source src="audio.wav" type="audio/wav"

<source src="audio.mp3" type="audio/mpeg"

For video elements, you code the same way as an audio element: <audio src=""></audio> but replace audio with video. The loop, controls and muted attributes all apply to video as well. If you wanted to display an image while the video is downloading, you can use the poster attribute.
